Prism Silverlight 4.0 Module Project
------------------------------------

Next Steps:
Update your Web project to add Silverlight application references to this Module project:
        * Right click on the Web project and select Project Properties
        * Click on the Silverlight Applications tab.
        * Click the Add button and select this Module project from the dropdown.
        * Click the Add button.
Update your Shell project's ModuleCatalog.xaml file to load this Module into the Shell.

Note:
Prism 4.0 provides signed binary assemblies. To ensure that project references are resolved correctly,
register the Prism assemblies with Visual Studio by running the RegisterPrismBinaries batch file.

For the latest information on Prism, see www.codeplex.com/prism.
